Customer Segmentation

In this project, I used different clustering algorithms (KMeans, KMedoids, Hierachical clustering agglorithms like agglomerative clustering, DBSCAN and Gaussian Mixture Model (GMM) to segement over 2000 customers whose insights and observations were later used to recommend market strategies that could help a company better utilize its resources and increase its return on investments.

Using the silhoutte score, the KMeans algorithm performed better than all the other algorithms and based on the output of the algorithm, methods like targeted advertising to different customers based on observed traits were recommended.

Used Car Price Prediction

In this project, I used different Machine Learning algorithms like linear regression, Decision trees, Random forests, KNN and boosting algorithms like ADAboost, Gradient boost, and XGBoost to predict future prices of over 6000 cars. Due to the finanacial incapability of customers to buy new cars on the market, the demand of used cars has been on a rise and therefore the urgent need of a used car price prediction algorithm which can effectively determine the worthiness of a car thus the aim of the project.

The Models used were back tested with the RMSE performance metric and were comapared with the R sqaured metric and the best model was observed to effectively predict future car prices with an 89% accuracy.

The Boston Housing Project

In this project, I used Exploratory data analysis and other machine learning techniques like linear regression to predict the future prices of over 500 houses in Boston. The aim for this project was for real estate customers to efficiently predict house prices with respect to their budgets

The linear model used in the prediction were back tested for over and under-fitting using the RMSE and MAE performance metrics and R-squared metric was employeed to compare the performance of the used models with the best model predicting houses with a 76% accuracy.

The Amazon Recommendation System

In this project, I build recommendation systems using Rank-based using average, User-User similarity-based, item-item similarity-based, and model-based collaborative filtering algorithms. The aim for this project was to accurately provide personalized recommendations to over 65000 customers on products that were likely to be interesting to them.

The models were back tested for under and over-fitting using RMSE. Different hyper-tuning techniques were employed to improve the performance of the models used and the best model was observed to recommend relevant products to customers with an 87% accuracy.
